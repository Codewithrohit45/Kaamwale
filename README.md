# 🚀 KaamWale

### Direct Local Workforce Hiring Platform

KaamWale is a full-stack MERN workforce hiring platform focused on rural and semi-urban India where customers can directly search, contact, and hire skilled workers like electricians, plumbers, carpenters, painters, mechanics, labour workers, cleaners, and many more without any middleman commission system.

---

# 🌟 Features

## 🔐 Authentication System
- User Signup/Login
- JWT Authentication
- Password Hashing
- OTP Email Verification
- Protected Routes

---

## 👥 User Roles
- Customer
- Worker
- Admin

---

## 🔍 Smart Search System
Search workers by:
- Category
- City
- Area
- Pincode
- Availability
- Ratings
- Pricing

---

## 👷 Worker Profiles
Each worker profile contains:
- Skills
- Experience
- Ratings & Reviews
- Pricing
- Availability
- Work Portfolio
- Verification Badge

---

## ⭐ Rating & Review System
Customers can:
- Rate workers
- Write reviews
- Provide feedback

---

## 📅 Booking System
Customers can:
- Book workers
- Choose date & time slots
- Add work description
- Track booking status

Workers can:
- Accept or reject bookings

---

## 🔐 OTP-Based Work Completion
After work completion:
1. Customer approves the work
2. OTP generated
3. Worker enters OTP
4. Booking marked completed
5. Payment released

---

## 🚨 Emergency Booking
Users can create urgent requests for:
- Electricians
- Plumbers
- Mechanics
- Repair services

---

## 👷 Bulk Labour Hiring
Hire:
- Multiple labour workers
- Construction workers
- Event staff
- Farming workers

---

## ❤️ Saved Workers
Users can save favourite workers.
